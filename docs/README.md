# Documentation

This directory contains marketing and documentation materials for the ClaimUI mod.

## Files

- **MODDB.md** - Mod DB listing information (summary and description)
- **images/** - Screenshots and promotional images

## Screenshots

Place screenshots in the `images/` directory with descriptive names:
- `claimui-ingame.png` - Main UI screenshot
- `claimui-demo.gif` - Usage demonstration (if available)

## Usage

When updating the Mod DB listing or creating promotional materials, refer to `MODDB.md` for the official descriptions and messaging.
