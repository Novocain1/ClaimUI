# Mod DB Listing Information

This file tracks the information used for the Vintage Story Mod DB listing.

## Summary (100 characters max)

A radial GUI for managing land claims with quick access to all claim commands via hotkey.

**Character count:** 94/100

## Description (HTML Format)

```html
<p><strong>ClaimUI</strong> is a client-side mod that provides a streamlined GUI interface for Vintage Story's land claim system. No more typing commands - just press <strong>L</strong> to open a radial button menu with instant access to all claim operations.</p>

<h3>Features</h3>
<ul>
<li><strong>Quick Access</strong>: Press <code>L</code> to toggle the claim UI (customizable in game settings)</li>
<li><strong>Radial Layout</strong>: All claim commands accessible through an intuitive button interface</li>
<li><strong>Smart Persistence</strong>: GUI automatically reopens when you restart the game</li>
<li><strong>Grow/Shrink Mode</strong>: OP button toggles between expanding and contracting your claims</li>
</ul>

<h3>Available Commands</h3>
<p>The UI provides instant access to:</p>
<ul>
<li><strong>New</strong> - Create a new claim</li>
<li><strong>Start</strong> - Begin claim selection</li>
<li><strong>End</strong> - Finalize claim selection</li>
<li><strong>Add</strong> - Add area to existing claim</li>
<li><strong>Cancel</strong> - Cancel current operation</li>
<li><strong>Save</strong> - Save claim data</li>
<li><strong>U/D/N/S/E/W</strong> - Grow or shrink claim in any direction</li>
<li><strong>List</strong> - List all your claims</li>
<li><strong>Load</strong> - Load most recent claim for editing</li>
<li><strong>OP</strong> - Toggle between grow and shrink modes</li>
</ul>

<h3>Installation</h3>
<ol>
<li>Download <code>ClaimUI.zip</code></li>
<li>Extract to your Vintage Story mods folder:
<ul>
<li><strong>Windows:</strong> <code>%APPDATA%\VintagestoryData\Mods\</code></li>
<li><strong>Linux:</strong> <code>~/.config/VintagestoryData/Mods/</code></li>
</ul>
</li>
<li>Restart Vintage Story</li>
<li>Press <code>L</code> in-game to open the Claim UI</li>
</ol>

<h3>Source Code</h3>
<p>Full source code available on GitHub: <a href="https://github.com/Novocain1/ClaimUI">Novocain1/ClaimUI</a></p>

<h3>Credits</h3>
<p>Original implementation by Novocain as part of the VSHUD mod. This standalone version extracts just the Claim UI functionality for players who want a lightweight land claim interface.</p>

```
