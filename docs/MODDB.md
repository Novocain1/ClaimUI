## Description

**ClaimUI** is a client-side mod that provides a streamlined GUI interface for Vintage Story's land claim system. No more typing commands - just press **L** to open a radial button menu with instant access to all claim operations.

### Features

- **Quick Access**: Press `L` to toggle the claim UI (customizable in game settings)
- **Radial Layout**: All claim commands accessible through an intuitive button interface
- **Smart Persistence**: GUI automatically reopens when you restart the game
- **Grow/Shrink Mode**: OP button toggles between expanding and contracting your claims

### Available Commands

The UI provides instant access to:
- **New** - Create a new claim
- **Start** - Begin claim selection
- **End** - Finalize claim selection
- **Add** - Add area to existing claim
- **Cancel** - Cancel current operation
- **Save** - Save claim data
- **U/D/N/S/E/W** - Grow or shrink claim in any direction
- **OP** - Toggle between grow and shrink modes

### Installation

1. Download `ClaimUI.zip`
2. Extract to your Vintage Story mods folder:
   - **Windows:** `%APPDATA%\VintagestoryData\Mods\`
   - **Linux:** `~/.config/VintagestoryData/Mods/`
3. Restart Vintage Story
4. Press `L` in-game to open the Claim UI
